%% Abstract
\begin{abstract}
\noindent
Can a neural network reliably predict which supply-chain relationships
break following climate shocks?  We test this question using 4,188
customer-quarter observations from Chinese A-share listed firms (2011--2023),
where the prediction target equals one when the cumulative net
termination measure over the next four quarters is positive
($\textit{BreakRisk4Q} = \mathbf{1}\!\{\sum_{h=1}^{4}\textit{Ter\_q\_qh}>0\}$).
A two-hidden-layer multilayer perceptron (MLP) trained on pre-2022 data
achieves holdout AUC of 0.536 on the 2022--2023 test period; a logistic
regression benchmark scores 0.544.  Rolling one-year-ahead AUCs range from
0.516 to 0.578 across five evaluation windows, indicating that limited
long-horizon predictability is a structural limitation of this task rather
than an artifact of the test window.  Shorter horizons are more predictable:
the MLP AUC is 0.668 for one-quarter disruption and 0.656 for two-quarter
disruption.  Applying a normalized-parameter-distance stability framework, we
show that successive MLP parameter vectors remain below the 95th-percentile
null threshold from same-sample random-seed perturbations (maximum observed
distance 0.49, threshold 1.53).  These results show that neural networks can
be structurally stable even when long-horizon discrimination is weak, and that
supply-chain climate-risk prediction is most useful at short horizons where
observed financial and climate features retain predictive content before
managerial adaptation and supplier substitution erode the signal.

\medskip\noindent
\textbf{Keywords:} climate risk; supply chain disruption; neural networks;
model stability; Chinese listed firms; machine learning; computational economics

\medskip\noindent
\textbf{JEL Codes:} C45, C53, G32, Q54
\end{abstract}

\section{Introduction}
%%==========================================================================

Can a neural network forecast which supply-chain relationships break after
a climate shock?  In our Chinese listed-firm sample the answer is only
partly yes.  A two-layer MLP is structurally stable across rolling samples,
but its 2022--2023 holdout AUC is 0.536---slightly \emph{below} a logistic
regression benchmark (AUC~$=$~0.544)---and rolling one-year-ahead AUCs
fluctuate between 0.516 and 0.578, never exceeding 0.60.  Yet the same
architecture predicts shorter-horizon disruptions more effectively: AUC is
0.668 for one-quarter disruption and 0.656 for two-quarter disruption.  We
argue that this horizon gradient is informative.  It suggests that observable
financial, governance, and climate variables contain short-run warning signal,
while longer-run relationship reorganization depends more on unmeasured
contractual terms, supplier substitution, and managerial responses that are
not recorded in mandatory disclosures.

The data come from mandatory disclosures of Chinese A-share listed firms,
which record bilateral customer-supplier relationships at a quarterly
frequency \citep{original2024climate}.  We construct a four-quarter
forward-looking disruption indicator based on the net change in relationship
terminations, and train a two-hidden-layer MLP on 17 features covering
upstream climate exposure, supplier financing constraints, customer
financials, governance characteristics, and seasonal controls.  The
training-test split is strictly temporal: no future data enter the training
set at any evaluation point.

The main predictive finding is that neither model provides strong
discrimination for the four-quarter target.  The MLP AUC of 0.536 and average
precision of 0.598 are broadly matched by logistic regression
(AUC~$=$~0.544, AP~$=$~0.591).  Extended model comparison shows that ridge
logit, random forest, and histogram gradient boosting also remain near this
range.  The important positive result is horizon dependence: MLP AUC falls
from 0.668 at one quarter to 0.536 at four quarters.  The AUC decline
reflects a progressive loss of feature informativeness as the forecast horizon
extends and firms adapt to initial shocks---not instability in the neural
network's learned parameters.  Permutation importance shows that customer
size, seasonal timing, and customer age dominate the four-quarter signal,
while upstream climate exposure and customer-city extreme precipitation
contribute positively but at a secondary level.

The main stability finding is that all five rolling MLP parameter vectors
lie well below the stability threshold.  Normalized Frobenius distances
from the 2018 reference model grow from 0.337 to 0.490 as training data
accumulate, but the 95th-percentile null threshold constructed from 20
same-data random-seed perturbations is 1.534---more than three times the
largest observed distance.  This means that the parameter shift attributable
to additional data is substantially smaller than the noise attributable to
random initialization on the same data.  The MLP architecture therefore
converges to a consistent set of parameters across sample expansions, even
though accuracy remains modest at four quarters.

This paper makes three contributions.  First, we apply the
normalized-parameter-distance stability framework of
\citet{stability2024framework} to supply-chain climate risk prediction,
showing how structural stability can be certified in a setting where
predictive accuracy is inherently limited.  Second, we document that static
financial and climate variables provide only weak discriminative power for
four-quarter bilateral relationship termination but stronger signal for
one- and two-quarter disruption, quantifying how predictability decays with
the forecast horizon.  Third, we show
through risk-surface analysis that the MLP reveals model-implied
associations between upstream climate exposure and supplier financing
constraints that the simple logistic benchmark does not represent without
explicit nonlinear terms---associations that may help prioritize further
diagnostic analysis, though they should not be read as causal mechanisms.

The remainder of the paper is organized as follows.  Section~\ref{sec:lit}
reviews related literature.  Section~\ref{sec:data} describes the data and
variable construction.  Section~\ref{sec:method} presents the model
architecture and stability framework.  Section~\ref{sec:results} reports
empirical results.  Section~\ref{sec:discussion} discusses implications and
limitations.  Section~\ref{sec:conclusion} concludes.

\section{Related Literature}\label{sec:lit}
%%==========================================================================

\subsection{Climate Risk and Corporate Supply Chains}

Aggregate climate events propagate through firm networks.
\citet{acemoglu2012network} showed that shocks to individual firms ripple
through production linkages to generate aggregate fluctuations, and
\citet{carvalho2021supply} confirmed this mechanism empirically using the
2011 Great East Japan Earthquake.  For climate risk specifically,
\citet{dietz2016climate} and \citet{battiston2017climate} quantified the
financial exposure of lending portfolios, while \citet{giglio2021climate}
surveyed the growing literature on physical and transition risk in asset
markets.  \citet{hong2019climate} found that climate risks are not yet
fully priced into Chinese equities.

At the firm level, \citet{hendricks2005empirical} documented persistent
declines in shareholder value following supply-chain disruptions.
\citet{original2024climate} established a causal link between extreme
precipitation and supply-chain reorganization among Chinese listed firms,
providing the data and motivation for our prediction exercise.  Our paper
asks a different question from that causal analysis: not \emph{why}
relationships break but whether the outcome can be predicted reliably in
advance from observable covariates.  The answer---that it can only partially
be predicted---highlights the gap between average causal effects and
individual-level predictability.

\subsection{Machine Learning for Financial and Supply-Chain Prediction}

\citet{gu2020empirical} compared machine-learning methods for equity return
prediction and found that neural networks improve over linear models when
nonlinearity is present.  Their evaluation protocol---rolling out-of-sample
windows with permutation-based feature importance---is the template we
follow.  \citet{altman1994neural} and \citet{khandani2010consumer}
established that neural networks can improve on discriminant analysis
for bankruptcy and consumer credit risk, in settings where the signal
is stronger than ours.

Machine-learning approaches to supply-chain risk prediction remain less
developed, partly because bilateral relationship-level data are scarce.
The Chinese mandatory-disclosure environment provides a rare panel, enabling
the prediction exercise we carry out.  Our results illustrate a case where
the prediction task is harder than standard credit risk: the observable
predictors leave a large unmodeled residual, so that an MLP and a logit
perform comparably.

\subsection{Neural Network Stability and Reliability}

The universal approximation theorem \citep{hornik1989multilayer} says that
MLP networks can represent any continuous function, but it says nothing about
whether gradient-based training produces stable solutions as the sample
grows.  \citet{stability2024framework} propose measuring neural network
stability via normalized Frobenius distances between successive parameter
vectors, with same-data random-seed perturbations providing the null
distribution.  We apply this test to a new domain and show that it can
certify stability even when predictive accuracy is low---which is precisely
the regime that makes the test most useful.

Related evaluation ideas appear in \citet{diebold1995comparing} on
predictive accuracy comparisons and in \citet{breiman2001random} on
solution stability in ensemble methods.  The key conceptual point, common to
all of these, is that stability and accuracy are separate properties: a model
can be structurally stable yet carry weak signal, or highly accurate yet
fragile to sample variation.

\subsection{Chinese Supply Chains and Financing Constraints}

China's disclosure requirements for major customer and supplier
relationships provide unusually granular data on bilateral supply-chain
links \citep{original2024climate}.  \citet{love2003financial} show that
trade credit is especially important where formal financial intermediation
is weaker.  This motivates including supplier financing constraints as a
predictor, while the disruption interpretation comes from the climate
supply-chain setting in \citet{original2024climate}.  In our data, the
interaction between upstream financing constraints and climate exposure
shapes model-implied disruption risk in ways that the simple logit
benchmark does not represent without added nonlinear terms.  Governance
characteristics---board independence, shareholder tunneling---appear in the
permutation-importance ranking, consistent with the literature on Chinese
trade credit that includes these controls.

\section{Data and Variable Construction}\label{sec:data}
%%==========================================================================

\subsection{Data Source and Sample Construction}

Our analysis uses the supply-chain reorganization panel constructed by
\citet{original2024climate} from mandatory disclosures of Chinese A-share
listed firms.  The dataset identifies bilateral customer-supplier
relationships at a quarterly frequency and records net relationship
terminations.  We restrict to observations for which all four forward
termination indicators ($\textit{Ter\_q\_q1}, \ldots, \textit{Ter\_q\_q4}$)
are available and all 17 prediction features are non-missing, yielding a
final sample of \textbf{4,188 customer-quarter observations} spanning
2011--2023.  The sample grows markedly over time---from fewer than 200
observations per year before 2015 to more than 800 in 2021--2022---reflecting
expanding A-share disclosure requirements (Figure~\ref{fig:sample}).

\subsection{Prediction Target}

The prediction target is:
\begin{equation}
\textit{BreakRisk4Q}_{i,t}
= \mathbf{1}\!\left\{
    \sum_{h=1}^{4} \textit{Ter\_q\_q}h_{i,t} > 0
  \right\},
\label{eq:target}
\end{equation}
where each $\textit{Ter\_q\_qh}_{i,t}$ is the net change in the number of
terminated focal-firm relationships for customer $i$ in quarter $t+h$.
The sum can take negative values (net new relationships) or positive values
(net terminations), so the indicator equals one when the cumulative net
termination measure over the next four quarters is positive---not simply
when any single relationship ends.  This construction captures a twelve-month
horizon relevant for procurement and financing decisions.

The observed positive-label rate averages about 49\% across the full
sample and is 56.0\% in the 2022--2023 holdout, reflecting elevated
supply-chain stress in that period.  A majority-class classifier would
achieve precision near 0.56 with no predictive information; useful
predictions must discriminate among firms within this disruption-prone
environment.

\subsection{Features}

Table~\ref{tab:features} lists the 17 input features in four groups.

\paragraph{Climate and Upstream Exposure.}
\emph{Affected} measures the customer's exposure to upstream focal firms
that simultaneously faced financing constraints \emph{and} extreme
precipitation shocks during the observation period, following the
construction in \citet{original2024climate}.  It is the key exposure
measure: it captures the double-stress condition that the original paper
identifies as the primary mechanism for forced supply-chain reorganization.
\emph{Climate\_CityC} is the log intensity of extreme precipitation in the
customer's registered city.  \emph{Affected\_x\_S\_FC} is the interaction
of the upstream exposure measure with the supplier financing constraint.

\paragraph{Supply-Chain Characteristics.}
\emph{S\_FC} (supplier financing constraint) measures the degree to which
the firm's key suppliers face binding credit constraints.
\emph{C\_Duration} captures relationship length in quarters.

\paragraph{Customer Financial Characteristics.}
Five accounting variables describe the customer firm: \emph{C\_Size} (log
total assets), \emph{C\_Lev} (leverage ratio), \emph{C\_Inv} (inventory
intensity), \emph{C\_Ppe} (fixed-asset intensity), and \emph{C\_Firmage}
(firm age since listing).

\paragraph{Governance and Temporal Variables.}
Governance controls are board size (\emph{C\_Board}), proportion of
independent directors (\emph{C\_Indep}), a shareholder-tunneling
indicator (\emph{C\_Occupy}), and customer-supplier transaction
frequency (\emph{C\_Frequency}).  Three temporal variables capture the
calendar year (\emph{year\_c}, demeaned) and within-year seasonality via
sine and cosine encodings of the quarter (\emph{quarter\_sin},
\emph{quarter\_cos}).

\subsection{Descriptive Statistics}

Figure~\ref{fig:sample} documents the prediction environment.  Panel A
shows the annual positive-label rate, which ranges from 0.42 to 0.58 with
no clear trend, indicating that net positive terminations are a persistent
feature of the sample rather than a spike concentrated in one period.  The
rate rises to 0.580 in 2022; this is a descriptive pattern rather than
evidence on the source of that year's supply-chain stress.

Panel B plots the annual means of the two climate variables.  The upstream
exposure variable (Affected) is highest in 2014--2016 (sample mean above
0.23) and falls to 0.085 by 2023.  The customer-city
extreme-precipitation variable (Climate\_CityC) peaks around 2016 and also
declines in 2023.  These time patterns motivate the temporal split but are
not, by themselves, evidence on policy or macroeconomic mechanisms.
Light bars flag years with fewer than 50 observations, where sample means
are less reliable.

\section{Neural Network and Stability Framework}\label{sec:method}
%%==========================================================================

\subsection{Multilayer Perceptron Classifier}

We train a two-hidden-layer MLP with 12 neurons in the first hidden layer
and 6 in the second, both with rectified linear unit (ReLU) activations
\citep{lecun2015deep}.  The output layer uses a logistic sigmoid to
produce a predicted disruption probability.  All inputs are standardized
to zero mean and unit variance before entering the network.  The (12,6)
specification is a deliberately low-capacity architecture: with 2,945
training observations and 17 input features, the 301 trainable parameters
yield a ratio of approximately ten observations per parameter.
Appendix~\ref{app:architecture} confirms that wider and deeper networks
(up to 3,265 parameters) do not materially improve four-quarter holdout AUC,
consistent with the view that feature informativeness rather than model
capacity is the binding constraint.

Training uses Adam stochastic gradient descent with initial learning rate
0.003 and L2 weight decay $\alpha = 0.01$.  We apply early stopping:
15\% of training observations form a validation set, and training halts
after 30 consecutive epochs without improvement in validation loss (maximum
1,500 epochs).  The random seed is fixed at 42 for all main estimates;
20 additional seeds (43--62) are used for the null distribution
(Section~\ref{subsec:stability}).  All estimation uses
\texttt{scikit-learn} \citep{pedregosa2011}.

The logistic regression benchmark uses the same feature set and
standardization pipeline, estimated by LBFGS (maximum 1,000 iterations).
It serves as a transparent linear reference to contextualize the MLP's
performance, not as a competing model to optimize.

\subsection{Train-Test Protocol}

\paragraph{Primary holdout.}
The model is trained on all observations with year $\leq 2021$ (2,945
observations) and evaluated on 2022--2023 (1,243 observations), mirroring
deployment conditions for a practitioner fitting the model at end-2021.

\paragraph{Rolling one-year-ahead evaluation.}
For training cutoffs 2018, 2019, 2020, 2021, and 2022, we train on all
observations through the cutoff year and evaluate on the following year.
We require at least 50 test observations and both outcome classes present
in train and test; all five windows satisfy these criteria.

\subsection{Evaluation Metrics}

We report: (i) AUC, the area under the ROC curve, which measures
rank-order discrimination independently of the decision threshold;
(ii) average precision (AP), which summarizes the precision-recall
trade-off; and (iii) Brier score, measuring squared probability error.
AUC is the primary metric because class balance fluctuates across windows
(observed rates 0.47--0.58).  AUC of 0.5 is the random-classifier
baseline; consistent values above 0.5 across all windows indicate that
observable features carry some forward-looking information.

\subsection{Stability Framework}\label{subsec:stability}

Following \citet{stability2024framework}, we assess structural model
stability by computing normalized parameter distances between successive
MLP estimates.  Let $\bm{\theta}_t$ be the vector formed by concatenating
all weight matrices and bias vectors of the MLP trained through year $t$.
Define the normalized Frobenius distance from the 2018 reference model:
\begin{equation}
d_t = \frac{\|\bm{\theta}_t - \bm{\theta}_{2018}\|}{\|\bm{\theta}_{2018}\|},
\label{eq:distance}
\end{equation}
where $\|\cdot\|$ denotes the Euclidean norm of the vectorized parameters.

The null distribution is constructed by training 20 additional MLP models
on the \emph{same} 2018 reference data with different random seeds (43--62)
and recording each model's distance from the reference.  The 95th percentile
of these 20 distances, $q_{0.95}^{\text{null}}$, is the stability threshold.
If $d_t < q_{0.95}^{\text{null}}$ for all $t$, the parameter variation
attributable to new data is no larger than the variation attributable to
random initialization on the same data; we describe this as structural
stability.

Passing the threshold does not imply high accuracy.  It implies that
the model's internal representation of the prediction problem is
consistent across sample expansions---a prerequisite for interpreting
historical performance as evidence about future behavior.

\subsection{Feature Importance}

We assess feature contributions via permutation importance
\citep{breiman2001random}.  For each feature $j$ in the holdout test set,
we permute its values 50 times, record the AUC on each permuted dataset,
and compute the mean AUC loss relative to the original.  Large positive
AUC loss indicates meaningful discriminative contribution; negative values
indicate features that add no useful information conditional on the rest.
Error bars report 95\% simulation intervals across permutations.

\subsection{Risk Surface Analysis}

To visualize the model's nonlinear predictions we compute risk surfaces
over the 2022--2023 holdout covariate distribution.  For Panel A of
Figure~\ref{fig:surface}, we vary upstream climate exposure (Affected)
from 0 to 1 across an evenly spaced grid, fix supplier financing
constraints at three representative levels
($\textit{S\_FC} \in \{0, 0.5, 1.0\}$), update the interaction term
consistently, and average predicted probabilities over the observed
distribution of all remaining features.  Panel B sweeps the
customer-city extreme-precipitation variable (Climate\_CityC) across
its empirical support while marginalizing over other features.

These surfaces show model-implied associations in the holdout data.
They are descriptive: they summarize how the trained model responds to
the two climate-related inputs and should not be interpreted as
estimates of causal effects.

\section{Empirical Results}\label{sec:results}
%%==========================================================================

\subsection{Sample Facts}

Figure~\ref{fig:sample} and Table~\ref{tab:rolling} characterize the
prediction environment.  The 4,188 observations are concentrated in
2019--2023, consistent with the expanding A-share disclosure coverage.
The positive-label rate is near-balanced in every evaluation window
(0.42--0.58), so neither class dominates.  The two climate variables trend
downward in the early 2020s; whether this reflects policy, reporting,
industrial adjustment, precipitation patterns, or sample composition cannot
be separated in this prediction exercise.  Most observations carry moderate
rather than extreme climate readings.

\subsection{Out-of-Sample Predictive Performance}

Table~\ref{tab:holdout} reports holdout metrics.  The MLP achieves AUC
\textbf{0.536}, average precision \textbf{0.598}, and Brier score
\textbf{0.252}.  Logistic regression yields AUC 0.544, AP 0.591, and Brier
0.250.  The two models perform comparably across all three metrics, with
logistic regression marginally ahead on AUC and Brier score and the MLP
marginally ahead on average precision.  Neither model provides strong
discrimination, and the nonlinear MLP architecture offers no decisive
advantage over the linear benchmark on this feature set.

Figure~\ref{fig:prediction} shows the ROC and precision-recall curves.
Both curves lie modestly above the diagonal (Panel A), and both
precision-recall curves barely exceed the no-skill line at the observed
rate of 0.560 (Panel B).  The near-overlap of the MLP and logit curves in
both panels suggests that the nonlinearity in the MLP does not translate
into a meaningful predictive gain in this application.

\paragraph{Interpretation.}
The modest AUC is not a failure of the estimation procedure.
Table~\ref{tab:rolling} shows that rolling one-year-ahead AUCs fluctuate
between 0.516 and 0.578 across five separate evaluation years, never
reaching 0.60.  Bootstrap confidence intervals (Appendix~\ref{app:bootstrap},
Table~\ref{tab:bootstrap}) sharpen this picture: only the
2020$\to$2021 window achieves a 95\% CI that clearly excludes 0.5
(AUC~$=$~0.578, CI~$[$0.542, 0.619$]$).  The remaining four windows return
confidence intervals that include 0.5, so predictive discrimination above
random chance cannot be established at the 5\% level in most evaluation
periods.  The AUC decline from 0.668 at one quarter to 0.536 at four quarters
indicates that observed features lose predictive content as the horizon
extends and firms adapt to initial shocks.  Supply-chain disruption
decisions reflect unobservable contractual terms, relationship-specific
dynamics, and managerial responses that cross-sectional static observables
only partially capture.

Additional robustness checks refine this conclusion.  Appendix~\ref{app:model}
shows that ridge logit, random forests, and histogram gradient boosting do not
materially improve four-quarter holdout AUC; their AUCs range from 0.528 to
0.544.  Appendix~\ref{app:ablation} shows that finance variables carry more of
the available four-quarter signal than climate-only or governance-only
features.  Appendix~\ref{app:horizons} shows that predictability is sharply
horizon-dependent: the MLP AUC is 0.668 for a one-quarter target and 0.656 for
a two-quarter target, falling to 0.536 for the four-quarter target used in the
main analysis.  Appendix~\ref{app:architecture} shows that changing MLP width
and depth does not remove the four-quarter performance ceiling.
Table~\ref{tab:summary} consolidates these findings: across all alternative
classifiers and architecture specifications, no configuration exceeds AUC 0.557
for the four-quarter target, while the same MLP reaches 0.668 at one quarter.

\subsection{Horizon-Dependent Predictability: Economic Interpretation}\label{subsec:horizon}

The weak four-quarter result masks substantial horizon dependence.
Table~\ref{tab:appendix_horizons} shows that the same MLP reaches AUC 0.668
for one-quarter disruption and 0.656 for two-quarter disruption, before
falling to 0.607 at three quarters and 0.536 at four quarters.  At the
one-quarter horizon the MLP also outperforms the logistic benchmark
(AUC~$=$~0.624), whereas at four quarters the benchmark is marginally ahead.
The horizon gradient is the paper's strongest positive result.

The decline in AUC from 0.668 to 0.536 is economically plausible.
One-quarter disruptions are temporally close to the realized shock and
therefore retain information from observable financial stress, supplier
financing constraints, and local climate exposure.  The prediction problem
at one quarter is largely about identifying which firms are currently fragile
in ways that accounting and climate disclosures capture with reasonable
accuracy.

Four-quarter relationship reorganization is a qualitatively different
prediction problem.  After the initial shock, firms can renegotiate contracts,
draw down inventory buffers, switch to alternative suppliers, or receive
financing support from affiliated or government-connected creditors.  These
adaptation responses are largely unobserved in the mandatory disclosure data:
bilateral transaction volumes, credit terms, network centrality, and inventory
drawdown plans are not disclosed at quarterly frequency.  The AUC decline from
0.668 to 0.536 therefore reflects a progressive loss of feature informativeness
as managerial adaptation and network substitution unfold---not instability in
the neural network's learned parameters, which remain consistently below the
null threshold throughout.  The architecture is stable; the four-quarter feature
set is weak.

This horizon gradient has practical implications for model deployment.  A
practitioner seeking to flag imminent supply-chain stress would find the model
most useful at one- and two-quarter horizons, where the MLP achieves AUC above
0.65 and outperforms the logistic benchmark.  For longer-run relationship
reorganization forecasting, the model adds little beyond the logistic baseline,
and richer data on bilateral transactions, supplier substitutability, and
network topology would be needed to improve discrimination.

\subsection{Calibration and Residual Diagnostics}

Figure~\ref{fig:calibration} presents two diagnostic panels.

Panel A is a calibration plot: observations are binned into deciles of
predicted probability and the observed positive-label rate per bin is
plotted against the mean predicted probability.  The positive but imperfect
slope---observed rates rise from 0.42 in the lowest decile to 0.60 in the
highest, while predicted values range from 0.29 to 0.72---confirms
directional information despite weak overall discrimination.  Calibration
is imperfect in the middle deciles, where predictions near 0.50 yield
observed rates between 0.49 and 0.56.

Panel B plots the autocorrelation function (ACF) of quarterly mean
residuals from the rolling predictions.  Most lags fall within the
$\pm 1.96/\sqrt{T}$ band.  The notable exception is lag 7 (ACF~$= 0.31$),
suggesting an annual or semi-annual structure in the disruption process
not captured by the quarter-cycle encoding.  Lags 1--5 show mild negative
autocorrelation ($-0.07$ to $-0.13$), consistent with slight mean reversion
in prediction errors.  The absence of strong positive autocorrelation at
short lags indicates the model does not systematically over- or
under-predict in consecutive periods.

\subsection{Model Stability}

Figure~\ref{fig:stability} and Table~\ref{tab:stability} report stability
diagnostics.  Panel A plots $d_t$ alongside the null threshold
$q_{0.95}^{\text{null}} = 1.534$.

Every $d_t$ is well below the threshold.  The distances increase
monotonically from 0.337 (cutoff 2019) to 0.490 (cutoff 2022), reflecting
that added data shift parameters away from the 2018 reference.  But the
largest distance (0.490) is less than one-third of the null threshold
(1.534).  For comparison, the 20 same-data null models yield distances of
1.27--1.66---all substantially above the cross-period distances.  The
parameter shift attributable to new data is therefore smaller than the
noise from changing the random seed on the same data: evidence of parameter
stability.

Panel B shows rolling one-year-ahead AUC, which fluctuates between 0.516
and 0.578 with no monotone trend.  The stable parameters in Panel A do
not translate into improving accuracy in Panel B, which is consistent with
the interpretation that the feature set constrains accuracy, not the model
architecture.

\paragraph{Note on interpretation.}
The stability result says that the model's learned association between
features and disruption risk does not fundamentally change as the training
sample grows.  This allows predictions from models trained in different
periods to be compared on a like-for-like basis.  It does not imply that
predictions are accurate; it implies that whatever inaccuracy exists is
stable and predictable in character.

\subsection{Feature Importance}

Figure~\ref{fig:importance} reports permutation importance.  The top
contributors by mean AUC loss are:
\begin{enumerate}
  \item Customer size (C\_Size): AUC loss 0.023---the dominant predictor
    by a wide margin.
  \item Quarter sine (quarter\_sin): AUC loss 0.012.
  \item Customer age (C\_Firmage): AUC loss 0.009.
  \item Quarter cosine (quarter\_cos): AUC loss 0.005.
  \item Relationship duration (C\_Duration): AUC loss 0.004.
\end{enumerate}

Upstream climate exposure (Affected) ranks sixth with AUC loss 0.004,
at the same magnitude as relationship duration.  Supplier financing
constraints (S\_FC) rank eighth (AUC loss 0.001), while the interaction
term and the city-level extreme-precipitation variable (Climate\_CityC)
produce near-zero or mildly negative contributions.

Several features---board size, fixed-asset intensity, inventory, calendar
year---show negative mean AUC losses, indicating that their permutation
coincidentally does not reduce discrimination given the dominant
predictors.

\paragraph{Interpretation.}
Customer size and age dominate, consistent with the finding that larger,
older firms maintain more stable supplier networks and can absorb
transitory shocks \citep{hendricks2005empirical}.  Seasonal patterns
reflect the business-cycle structure of Chinese manufacturing, where
supply-chain adjustments concentrate around fiscal year-end and major
holidays.  Relationship duration contributing positively is consistent
with the notion that established relationships are more resilient to
disruption.  The modest rank of upstream climate exposure (sixth) indicates
that cross-sectional size and age differences are stronger discriminators
of disruption risk than climate readings in any given year---not that
climate exposure is economically unimportant.

\subsection{Risk Surfaces}

Figure~\ref{fig:surface} shows model-implied risk surfaces.  These
curves summarize model-implied associations in the holdout data; they are
not causal estimates.

\paragraph{Panel A: Upstream exposure and financing constraints.}
At low financing constraints (S\_FC~$= 0$), predicted disruption risk
declines monotonically from 0.555 to 0.491 as upstream climate exposure
increases from 0 to 1.  The negative gradient may reflect that firms
with high upstream exposure had already restructured vulnerable
relationships before the holdout period, or that they operate in
industries with more substitutable suppliers.

At high financing constraints (S\_FC~$= 1.0$), the pattern is U-shaped:
risk is highest at zero upstream exposure (0.458), falls to a minimum near
Affected~$= 0.525$ (0.423), and rises gradually at higher exposure levels.
This non-monotone pattern is not represented in the simple logistic
benchmark without explicit quadratic or interaction terms.  A possible
interpretation is that moderately exposed, highly constrained firms occupy
a middle position where neither extreme vulnerability channel dominates,
though other explanations cannot be ruled out.  We report this as a
model-implied descriptive pattern, not a mechanism.

At intermediate constraints (S\_FC~$= 0.5$), the curve is approximately
flat, declining slightly from 0.497 to 0.461.

\paragraph{Panel B: Customer-city extreme precipitation.}
Predicted risk declines gradually from 0.544 at zero precipitation
intensity to 0.531 at the upper end of the observed distribution.  The
shallow slope is consistent with the near-zero permutation importance of
Climate\_CityC: city-level aggregate shocks affect all local firms
simultaneously and therefore provide little cross-sectional discrimination.

%%==========================================================================
\section{Discussion}\label{sec:discussion}
%%==========================================================================

\subsection{Why Limited Accuracy Is Informative}

The main finding---AUC consistently near 0.54 across both models and all
rolling windows---tells us something about the data-generating process.
It implies that the observable firm characteristics, governance indicators,
and climate variables in our feature set capture only part of the variation
in supply-chain disruption outcomes.  The unmodeled residual reflects
relationship-specific factors: informal trust and contractual lock-in that
are not disclosed, idiosyncratic production shocks, and managerial
responses that unfold after the observation date.  Richer data---bilateral
transaction volumes, credit terms, network centrality measures---would
likely improve the signal.

The stability result adds a complementary piece of information.  It shows
that what the model \emph{does} capture from these features is stable over
time: the architecture does not drift as training data grow.  The
combination of limited accuracy and structural stability is the expected
pattern when the feature set is informative but incomplete.  A feature set
that was truly uninformative would produce both low accuracy and unstable
parameters; a feature set that fully described the outcome would produce
both high accuracy and stability.  Our finding places this application
somewhere in between, with the important qualification that short-horizon
predictability substantially exceeds four-quarter predictability.

\subsection{The Role of Upstream Climate Exposure}

Permutation importance ranks upstream climate exposure (Affected) sixth,
with an AUC loss of 0.004.  Given that the total AUC above 0.5 is about
0.036, this represents roughly 11\% of the discriminative signal
($0.004 / 0.036$).  The contribution is small in absolute terms but not
negligible relative to the available signal.

The risk-surface analysis provides additional information about the
\emph{shape} of the climate association.  The non-monotone pattern at high
financing constraints (Panel A) suggests that the climate exposure--disruption
relationship is conditioned by financing conditions in a way that
permutation importance---which evaluates marginal contributions averaged
over the holdout---does not capture.  These patterns warrant further
investigation, but because the prediction task is underpowered and the
surface reflects model-implied associations rather than causal effects,
strong economic conclusions about the mechanism should be deferred to
studies with better identification.

\subsection{Implications for Model Deployment}

Our results suggest a practical evaluation protocol for deploying neural
networks in climate risk assessment.  First, report accuracy metrics
honestly across multiple evaluation windows, not only the best window.
Second, test structural stability before interpreting accuracy as evidence
about future performance.  Third, use permutation importance to check
whether the dominant predictors are economically interpretable.  Fourth,
use risk surfaces to look for model-implied heterogeneity that might
otherwise be obscured by aggregate metrics.

In our application, passing all four checks yields a consistent message:
the model is stable and its top-ranked predictors are economically
interpretable, but accuracy is limited at four quarters, and deploying it
for operational risk scoring requires acknowledging this limitation and
matching the forecast horizon to the decision context.

\subsection{Limitations}

Several limitations apply.  The sample covers one country and one
institutional context; the stability and importance findings may not extend
to markets with different disclosure rules or financial structures.  The
feature set is restricted to publicly observable accounting and climate
variables; bilateral transaction data would be needed to improve accuracy
substantially.  Extended model comparison and architecture-sensitivity checks
in the appendix do not reveal a flexible classifier that dominates the simple
logit or baseline MLP on the four-quarter target, though shorter horizons are
meaningfully more predictable.  Finally, the stability test we
apply compares cross-period distances to a same-data random-seed null; more
demanding tests could compare against out-of-distribution samples or
adversarial perturbations.

%%==========================================================================
\section{Conclusion}\label{sec:conclusion}
%%==========================================================================

We test whether a neural network can reliably predict climate-induced
supply-chain disruption risk using Chinese A-share data.  The answer has two
parts.  First, a two-layer MLP trained on pre-2022 data achieves holdout
AUC of 0.536 on the 2022--2023 test period, roughly matching a logistic
regression benchmark (AUC~$=$~0.544), with rolling one-year-ahead AUCs
ranging from 0.516 to 0.578 across five windows.  Neither model provides
strong discrimination.  Second, normalized-parameter-distance diagnostics
show that successive MLP parameter vectors remain consistently below the
95th-percentile null threshold (maximum distance 0.490, threshold 1.534),
providing evidence of structural stability independently of accuracy.

The limited predictive gain is itself informative: it indicates that
bilateral supply-chain disruption decisions contain a large
relationship-specific component not captured by static financial and
climate observables.  Bootstrap confidence intervals (Appendix~\ref{app:bootstrap})
further show that only one of five rolling evaluation windows achieves an
AUC confidence interval that clearly excludes 0.5; the predictive advantage
over random chance is statistically fragile.  The stability result separates
this data-informativeness limitation from a model-reliability question.  The
architecture is reliable in the sense that what it learns is consistent over
time; it is limited in the sense that the features it learns from do not
fully describe the outcome.

The horizon analysis provides the paper's strongest positive finding.
The MLP AUC declines from 0.668 at one quarter to 0.536 at four quarters,
reflecting a progressive loss of feature informativeness as managerial
adaptation and network substitution unfold.  The architecture is stable
throughout this gradient; the bottleneck is the four-quarter feature set,
not model capacity.  Unobserved contracts, supplier substitution, and
managerial adaptation matter more as the horizon extends.

Permutation importance identifies customer size, seasonal timing, and
customer age as the dominant predictors.  Upstream climate exposure
contributes positively but at a secondary level, consistent with the
conclusion that firm fundamentals are stronger cross-sectional discriminators
of disruption risk than climate readings.  Risk-surface analysis shows
model-implied associations between upstream climate exposure and supplier
financing constraints---including a non-monotone pattern at high constraint
levels---that the simple logistic benchmark does not represent, though these
should be read as descriptive patterns rather than causal mechanisms.

The broader takeaway for computational economics is methodological.  In
prediction tasks with unmeasured relationship-specific determinants,
documenting structural model stability and reporting accuracy across multiple
evaluation windows are both needed before interpreting practical utility.
The appendix results add one constructive qualification: supply-chain
disruption is much more predictable at one- and two-quarter horizons than at
the four-quarter horizon.  Future work should therefore enrich the feature set
with network-topology and bilateral transaction data, and distinguish more
carefully between short-run warning systems and longer-run relationship
reorganization forecasts.

Predictability is substantially stronger at short horizons.  The MLP reaches
AUC 0.668 for one-quarter disruptions and 0.656 for two-quarter disruptions,
but the AUC declines to 0.536 for the four-quarter target.  At one and two
quarters the MLP also outperforms the logistic benchmark, whereas at three
and four quarters the logit is marginally ahead.  The horizon gradient
reflects a progressive loss of feature informativeness as managerial
adaptation and network substitution erode the predictive content of static
observables over longer periods.

\section{Replication Note}\label{app:replication}
%%--------------------------------------------------------------------------

All results reported in this paper are produced using Python~3.  Machine
learning models use \texttt{scikit-learn} \citep{pedregosa2011}; data
handling uses \texttt{pandas} and \texttt{numpy}; the original Stata data
file is read via \texttt{pyreadstat}; figures are produced with
\texttt{matplotlib}.  Exact package versions are recorded in the
\texttt{requirements.txt} file included in the replication archive.

Random seeds are as follows: the main MLP and logistic regression estimates
use seed 42; the stability null distribution uses seeds 43--62; bootstrap
resampling in Appendix~\ref{app:bootstrap} uses seed 0 with $B=1{,}000$
draws.  The architecture sensitivity estimates in Appendix~\ref{app:architecture}
also use seed 42 for each specification.

To reproduce all results from a clean directory, place the original Stata
data file in the location specified in \texttt{appendix\_code/common.py}
and run the five replication scripts in sequence:

\begin{verbatim}
python appendix_code/appendix_a_bootstrap_ci.py
python appendix_code/appendix_b_model_comparison.py
python appendix_code/appendix_c_ablation.py
python appendix_code/appendix_d_horizons.py
python appendix_code/appendix_e_architecture.py
\end{verbatim}

Each script writes output to \texttt{appendix\_data/}.  No result reported
in the manuscript depends on code that has not been run or data that are not
generated by these scripts.
