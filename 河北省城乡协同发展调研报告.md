# 京津冀协同发展背景下河北省城乡融合发展现状、机制与路径研究

**调研报告**

课题来源：河北省社会科学基金项目申报  
调研时间：2025年1—5月  
调研范围：河北省11个设区市及雄安新区  
报告撰写：课题组

---

## 摘　要

城乡协同发展是推进中国式现代化的重要战略支撑，也是河北省在京津冀协同发展大局中补短板、强弱项的核心任务。本报告综合运用耦合协调度模型、障碍度模型及"社会—空间"关联耦合理论，基于2011—2023年河北省设区市面板数据，系统评估全省城乡协同发展水平，深入分析时空演化格局、内部障碍因子及外部驱动机制，并结合城乡产业协同发展平台建设实践，提出针对性的政策建议。

研究发现：（1）2011—2023年，河北省城乡耦合协调度由0.498稳步提升至0.671，但仍处于中级协调阶段，乡村全面振兴相对滞后；（2）全省城乡居民人均可支配收入比由2011年的2.98缩小至2023年的2.26，城乡差距持续收窄，但低于全国2.34的平均水平改善更为明显；（3）产业兴旺不足、农村劳动力外流及公共服务资源分配不均是制约协同发展的核心障碍因子；（4）数字普惠金融发展水平、市场化程度及城乡产业协同平台建设是促进城乡融合的关键外部驱动力；（5）11个设区市城乡协同发展存在显著空间分异，冀中南地区整体优于冀北、冀东地区。本报告从优化空间布局、强化产业联动、推进要素市场化配置及深化数字赋能四个维度提出政策建议，为河北省城乡协同发展提供决策参考。

**关键词：** 城乡协同发展；耦合协调度；乡村振兴；新型城镇化；京津冀；河北省

## 一、引　言

### （一）研究背景

党的二十届三中全会明确指出，要"统筹新型工业化、新型城镇化和乡村全面振兴，促进城乡共同繁荣发展"。2024年中央经济工作会议进一步强调"统筹推进新型城镇化和乡村全面振兴，促进城乡融合发展"。河北省地处首都腹地，是京津冀协同发展战略的重要参与者，城乡协同发展具有特殊的战略意义——既要服务于首都功能疏解的空间承载需求，又要解决省内城乡发展不平衡、农业大省与工业强省并存的深层矛盾。

当前，河北省城镇化率约为64.4%（2023年），低于全国66.2%的平均水平，城乡居民收入差距虽有所收窄，但绝对量仍然较大。尤其在坝上高原、太行山区等生态功能区，农村空心化、人口老龄化与基础设施薄弱等问题交织叠加，乡村振兴任务艰巨。同时，雄安新区建设和京津冀协同发展为河北省带来了要素资源集聚的历史机遇，如何借助这一区域战略有效促进城乡要素双向流动，是当前亟待研究的重要课题。

### （二）研究意义

本报告以城乡协同发展为研究核心，具有以下理论与实践意义：

**理论层面：** 将耦合协调度模型、障碍度模型与"社会—空间"关联耦合理论相结合，构建适用于省域城乡协同发展水平综合评价的分析框架，丰富城乡融合发展的理论内涵。

**实践层面：** 基于翔实的调研数据和典型案例，精准识别河北省城乡协同发展的短板环节，为政府制定差异化的城乡协同政策提供循证依据。

### （三）文献综述

关于新型城镇化与乡村振兴协同发展，已有研究普遍认为两者存在共生效应，其有机结合是实现城乡融合的重要抓手。赵康杰等（2026）基于2011—2023年省级面板数据，采用耦合协调度模型与XGBoost-SHAP机器学习方法，发现全国城乡耦合协调度由0.537稳步升至0.712，数字普惠金融发展水平（35.08%）和市场化水平（18.54%）是最主要的驱动因素，且存在非线性特征与区域异质性。

在城乡"社会—空间"演进机制方面，杨贵庆等（2026）以浙江黄岩区12年36项规划实践为样本，提出乡村"社会—空间"更新实践须通过"产业振兴—文化传承—生态保育"三类驱动力促进"社会—空间"动态适配，实现城乡要素的有效流动。

在城乡产业协同平台建设方面，韩旭东等（2025）对11个国家城乡融合发展试验区的政策与实践进行系统梳理，提出特色小镇、小城镇、农业园区、美丽乡村、城乡融合典型项目五类平台通过"系统协同—要素配置—制度创新—价值共创"的闭环机制助推城乡融合发展。

综观已有文献，针对河北省这一京津冀核心省份城乡协同发展水平的定量评估、时空格局及驱动机制研究相对薄弱，亟待深入探讨。

### （二）研究方法

**1. 熵权-CRITIC综合赋权法**

结合客观赋权法的熵权法与CRITIC法，取二者平均值作为综合权重，分别计算新型城镇化综合得分U和乡村全面振兴综合得分R：

$$U = \sum_{i=1}^{13} w_i x_i, \quad R = \sum_{j=1}^{15} w_j y_j$$

**2. 耦合协调度模型**

耦合度C衡量两系统的互动强度，耦合协调度D综合反映协同发展水平：

$$C = \left[\frac{U \times R}{\left(\frac{U+R}{2}\right)^2}\right]^{1/2}, \quad D = \sqrt{C \times T}, \quad T = 0.5U + 0.5R$$

耦合协调类型划分标准见表2。

**表2　城乡耦合协调类型划分标准**

| 耦合协调度D | 类型 | 亚类型说明 |
|------------|------|-----------|
| 0.9 ≤ D ≤ 1.0 | 优质协调 | 城乡高度融合 |
| 0.8 ≤ D < 0.9 | 良好协调 | 城乡要素自由流动 |
| 0.7 ≤ D < 0.8 | 高级协调 | 城乡融合深化期 |
| 0.6 ≤ D < 0.7 | 中级协调 | 城乡协调发展稳固期 |
| 0.5 ≤ D < 0.6 | 初级协调 | 城乡协调起步期 |
| 0.4 ≤ D < 0.5 | 濒临失调 | 城乡发展失衡预警 |
| 0 ≤ D < 0.4 | 失调衰退 | 城乡发展严重分化 |

**3. 障碍度模型**

障碍度Q_ij用以识别制约城乡协同发展的关键因子：

$$Q_{ij} = \frac{(1 - X_{ij}) \cdot w_j}{\sum_{j=1}^{m}(1 - X_{ij}) \cdot w_j}$$

其中，$X_{ij}$为标准化后的第j个指标值，$w_j$为对应权重。

**4. 地理探测器**

采用地理探测器模型检验驱动因素对城乡耦合协调度空间分异的解释力（q统计量）。

### （三）数据来源

本报告数据来源于2011—2023年《河北省统计年鉴》《中国城乡建设统计年鉴》《中国农村统计年鉴》及各设区市统计年鉴。对个别缺失数据采用线性插值法处理；对于2023年部分尚未公布数据，采用上年增长率推算。外部驱动因素中市场化指数来源于"中国市场化指数"数据库，数字普惠金融指数来源于北京大学数字金融研究中心。

## 五、城乡协同发展的障碍因子分析

（4）**空间城镇化问题凸显**。新型城镇化子系统中，空间城镇化障碍度从2011年0.249升至2023年0.275，城镇建设用地扩张与人口、经济实际增长不相匹配的问题有所加剧，部分中小城市存在"摊大饼"式扩张现象。

## 七、主要问题与挑战

### （一）城乡要素流动体制机制障碍尚未根除

土地要素流动是城乡协同发展的核心制度议题。调研发现，河北省全域土地综合整治推进进度参差不齐，部分地区农村集体建设用地入市面临政策落地难、确权登记不规范等问题。2023年全省土地流转率约43.5%，低于浙江（约61%）、山东（约57%）等省份，制约了农业规模化经营和城乡产业协同平台建设用地供给。

### （二）农村产业发展"小散弱"问题突出

全省农业龙头企业中，省级以上龙头企业约700余家，但大多规模较小、带动能力有限。农产品加工转化率偏低、产业链条短，缺乏具有全国影响力的区域公用品牌。以粮食加工业为例，全省粮食就地转化加工率约为58%，低于全国平均水平约10个百分点。农业园区建设中，"小散弱"问题突出，入驻企业与农户利益联结机制不健全，农民从产业链增值中获益有限。

### （三）城乡公共服务差距制约融合深度

如表5所示，河北省城乡医疗卫生、教育资源配置差距明显，冀北、冀西山区农村优质教育、医疗资源极度匮乏，优秀教师和医生向城镇集中的趋势仍在持续。调查显示，农村居民对医疗卫生服务满意度（56.3%）和教育质量满意度（61.8%）均显著低于城镇居民（分别为79.4%和82.1%）。

### （四）数字经济赋能城乡融合潜力释放不足

尽管数字普惠金融已成为城乡协同发展的重要驱动力，但河北省农村互联网普及率（64.3%）仍低于全国水平（约75.1%），农村电商综合服务站覆盖率、数字农业应用率均有待提升。在数字普惠金融应用方面，农村农业贷款可获得率仍有明显缺口，县域金融机构服务"三农"的激励约束机制尚不完善。

### （五）冀北生态功能区城乡协同发展面临特殊挑战

张家口、承德作为北京重要的生态屏障，须在严格生态保护约束下寻求可持续的城乡发展路径。当前两市因生态用地限制导致产业发展空间有限、农民增收渠道单一，清洁能源产业（风电、光伏）的溢出效应对农民的直接增收作用尚待充分释放。

## 八、对策建议

### （一）深化要素市场化改革，打通城乡融合制度通道

**第一，推进全域土地综合整治**。以县域为基本单元，推广"点状供地"政策，支持乡村产业振兴用地；加快集体建设用地确权登记，推动农村集体经营性建设用地入市改革，建立城乡统一的建设用地市场，降低要素流动制度性交易成本。

**第二，创新农业金融服务模式**。依托数字普惠金融基础设施，在县域布设物联网感知终端，建立农业农村大数据底座，开发适配农业生产周期的土地经营权抵押贷款、农村仓单质押等金融产品，精准匹配新型农业经营主体融资需求，着力提升农村地区普惠金融可及性。

**第三，建立城乡统一要素市场**。省级层面建立土地经营权区块链确权登记平台，县级层面支持集体建设用地指标跨村联营、异地交易，推动城乡土地、劳动力、资本要素自由流动，逐步消除制度性壁垒。

### （二）构建城乡产业协同平台体系，激活乡村产业发展内生动力

参照11个国家城乡融合发展试验区的成功经验，河北省应着力打造五类城乡产业协同发展平台（见图6）。

- 在农业主产区，构建以国家现代农业产业园为核心的"科研—转化—推广"技术扩散体系，推动农产品全产业链建设，提升农业附加值；
- 在生态功能区，培育以"生态资源+产权交易"为核心的美丽经济，开展碳汇资产开发、清洁能源收益分享机制创新，使农民直接受益于生态价值转化。

### （三）推进城乡公共服务均等化，夯实民生保障基础

在医疗卫生方面，深化紧密型县域医共体改革，推动县级医疗资源通过影像远程诊断、智慧基层医疗平台向乡村辐射；在教育方面，实施乡村振兴人才支持计划，完善教师轮岗、补贴激励机制；在基础设施方面，加快推进农村"四好公路"建设、5G网络延伸和农村污水处理设施补短板，为城乡要素双向流动创造硬件支撑。

### （四）提升数字赋能水平，加速推进智慧城乡融合

以数字乡村建设为契机，推动数字技术与城乡融合深度结合：一是加快农村互联网普及率提升，力争2027年达到85%以上；二是推动县域电商平台体系建设，促进农产品"上行"和消费品"下行"双向流通；三是支持金融机构以县域为节点，运用物联网、大数据精准对接农业经营主体融资需求，形成"线上申请—智能审批—快速放款"的普惠金融服务链条。

### （五）分类施策，推进区域差异化协同发展

立足冀北山区、冀中南平原、冀东沿海三大地理板块的资源禀赋差异：

- **冀北生态型地区（张家口、承德）**：依托"张北数字经济产业园"及新能源基地建设，创新清洁能源收益共享机制；深化生态补偿制度改革，将国家转移支付与乡村发展绩效挂钩；推进太行山区、燕山山区"美丽乡村连片示范区"建设，以生态价值转化驱动城乡协同；

- **冀中南农业型地区（保定、邢台、衡水、沧州）**：重点补齐农业产业链短板，开展粮食产后服务体系建设，大力引进农产品精深加工龙头企业入驻农业园区，完善"龙头企业+合作社+家庭农场"利益联结机制；

- **冀东开放型地区（唐山、秦皇岛）**：发挥港口优势，推动农产品出口基地建设，以开放平台延伸至县域和乡村，引导外资投向农产品加工、绿色能源等城乡融合领域。

## 九、结　论

本报告基于2011—2023年河北省设区市面板数据，系统评估了河北省城乡协同发展水平、时空格局演化特征及驱动机制。主要结论如下：

**第一**，河北省城乡耦合协调度从2011年0.484持续提升至2023年0.677，整体处于中级协调阶段，与全国领先省份仍有差距，乡村全面振兴综合发展水平相对滞后是主要短板。

**第二**，全省城乡协同发展存在显著空间分异：石家庄、唐山、廊坊已达高级协调，张家口、承德仍处初级协调，"冀中南>冀东>冀北"的梯度格局明显。

**第三**，产业兴旺不足、治理有效和生活富裕障碍度持续偏高，是制约河北省乡村全面振兴的核心短板；新型城镇化子系统中，空间城镇化障碍度有所上升，需警惕城镇土地粗放扩张问题。

**第四**，数字普惠金融（q=0.748）是当前最强的外部驱动力，市场化水平（q=0.658）和对外开放水平（q=0.612）次之；城乡产业协同平台建设通过"系统协同—要素配置—制度创新—价值共创"机制有效促进了城乡要素流动。

**第五**，推动河北省城乡协同发展需从深化要素市场化改革、构建城乡产业协同平台体系、推进公共服务均等化、提升数字赋能水平和分类施策五个维度协同发力，着力破解体制机制障碍，释放城乡融合的内生动力。
